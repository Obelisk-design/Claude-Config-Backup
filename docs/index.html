快速开始

1

创建 GitHub OAuth 应用

访问 GitHub 开发者设置，创建新的 OAuth 应用

  • Application name: Claude Config Backup
  • Homepage URL: http://localhost:18080
  • Authorization callback URL: http://localhost:18080/callback
2

配置应用

启动应用后，在"设置"标签页选择存储类型并填写相关配置，然后点击"保存设置"

3

开始使用

如果使用 GitHub 存储，点击"GitHub 登录"进行授权；其他存储类型可直接开始备份

敏感信息处理

⚠️
安全提示

备份时会自动识别并处理敏感信息，包括 API Token、密码、密钥等字段

处理方式：

  • 脱敏：将敏感值替换为 ***MASKED***
  • 排除：某些字段完全排除（如 ANTHROPIC_AUTH_TOKEN）

常见问题

?

登录失败怎么办？

  • 检查 GitHub OAuth 应用配置是否正确
  • 确认回调端口（18080）未被占用
  • 检查网络连接是否正常
?

如何在新设备恢复配置？

  • 在新设备安装并配置应用
  • 使用同一 GitHub 账号登录
  • 从"历史"标签选择备份恢复
💡
小贴士

恢复前建议勾选"备份当前配置"，以便出现问题时可以回滚
